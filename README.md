
# Design and Analysis of Algorithms (DAA) Lab

This repository contains my implementations of core algorithms in **C** as part of the Design and Analysis of Algorithms laboratory course. 

The primary focus is to analyze time and space complexities, understand different algorithmic design paradigms, and implement efficient solutions.
